# מדריך עדכון גרסה 2.4 - CRM וניהול לידים

גרסה זו כוללת מערכת ניהול לקוחות (Directory) וניהול לידים (Pipeline).

## 1. עדכון קוד גוגל סקריפט (חובה לסנכרון!)
מכיוון שהוספנו טבלה חדשה של **Leads** (לידים), צריך לעדכן את הסקריפט גם ב"הכנסות" וגם ב"הוצאות" (ליתר ביטחון, עדיף שבשניהם יהיה אותו קוד עדכני).

1.  פתח את הקובץ `קוד_סקריפט_גוגל.txt` מהתיקייה החדשה.
2.  העתק את כל הקוד.
3.  לך לגוגל שיטס של **הכנסות (Incomes)** > Extensions > Apps Script.
4.  הדבק את הקוד החדש.
5.  **חשוב:** בצע **Deploy New Version** (כפתור כחול -> Manage deployments -> Pencil -> New version -> Deploy).
6.  אין צורך לשנות את הקישור באפליקציה אם עשית New Version.

## 2. העלאת האתר
1.  העלה את כל הקבצים מתיקיית `WEBSITE_V2.4_FINAL` לגיטהאב.
2.  רענן את האתר ובדוק שכתוב למעלה **(v2.4)**.

## 3. שימוש בפיצ'רים החדשים
*   **CRM (טאב כחול):**
    *   **לידים (Pipeline):** ניהול מתעניינים. ניתן להוסיף ליד חדש, ולשנות סטטוס (חדש/בטיפול/סגור) בלחיצה על 3 הנקודות בכרטיס.
    *   **לקוחות:** רשימת כל הלקוחות. ניתן לחפש לפי שם/טלפון.
    *   **פעולות:** לחיצה על טלפון מחייגת, לחיצה על הודעה פותחת וואטסאפ.
    *   **LTV:** בכניסה ללקוח רואים סיכום של כל ההכנסות ממנו.
